# Vicon Path and VINS-Fusion path Alignment Tool
此工具為VINS-Fusion輸出軌跡與VICON等無時間截之參考軌跡資料對準與對時工具。

比較軌跡格式:['timestamp', 'px', 'py', 'pz', 'qw', 'qx', 'qy', 'qz', 'vx', 'vy', 'vz']

真實軌跡格式:['timestamp','px', 'py', 'pz', 'qw', 'qx', 'qy', 'qz' ]
